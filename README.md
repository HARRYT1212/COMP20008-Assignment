# COMP20008-Assignment
Whether a squirrel is likely to approach or avoid humans, and identifies which variables are most influential in predicting this behaviour.
# Assignment 2 Code

The main file to run is: Assignment 2 Code and Comments.ipynb

It combines all parts of the analysis in one notebook:

1. Data preprocessing
2. Correlation and feature analysis
3. KNN and decision tree models
4. Logistic regression model
5. Clustering analysis

## Files

- `Assignment 2 Code and Comments.ipynb`: final combined notebook
- `squirrel.csv`: original squirrel dataset
- `hectare.csv`: original hectare dataset
- `stories.csv`: original stories dataset
- `squirrel_processed.csv`: cleaned dataset created by the preprocessing section

## How To Run

1. Open `Assignment 2 Code and Comments.ipynb` in Jupyter Notebook, JupyterLab, VS Code, or another notebook editor.
2. Make sure the CSV files are in the same folder as the notebook.
3. Run all cells from top to bottom.

The notebook will first recreate `squirrel_processed.csv`, then use it for the analysis and models.

## Outputs

Running the notebook produces:

- cleaned dataset output: `squirrel_processed.csv`
- summary tables for preprocessing and feature analysis
- correlation, mutual information, and feature importance plots
- KNN, decision tree, logistic regression
- clustering tables and plots
